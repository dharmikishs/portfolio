# Portfolio Pro Shine

A modern, professional portfolio web application built with React, TypeScript, Vite, and Tailwind CSS.

## Features
- Responsive design
- Modular component structure
- Sections for About, Skills, Projects, Certifications, Experience, and Contact
- Custom UI components
- Easy to customize and extend

## Getting Started

### Prerequisites
- Node.js (v16 or higher recommended)
- npm or bun (for package management)

### Installation

1. Clone the repository:
   ```sh
   git clone <repository-url>
   cd portfolio-pro-shine-main
   ```
2. Install dependencies:
   ```sh
   npm install
   # or
   bun install
   ```
3. Start the development server:
   ```sh
   npm run dev
   # or
   bun run dev
   ```
4. Open your browser and visit [http://localhost:5173](http://localhost:5173)

## Project Structure
- `src/components/` - UI and section components
- `src/pages/` - Page components
- `src/hooks/` - Custom React hooks
- `src/lib/` - Utility functions
- `public/` - Static assets

## Customization
Edit the components in `src/components/` and content in `src/pages/` to personalize your portfolio.
